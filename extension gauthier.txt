# Extensions VS Code

> Attention, certaines nécessitent un environnement de développement déjà configuré
> pour pouvoir fonctionner !  
> Une mention `!!` sera ajoutée pour vous permettre d'être au courant.

## Thèmes et apparence d'interface

- Thème clair : [Atom One Light Theme](https://marketplace.visualstudio.com/items?itemName=akamud.vscode-theme-onelight)
- Thème sombre : [Theme](https://marketplace.visualstudio.com/items?itemName=tal7aouy.theme)
- Icônes : [Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=PKief.material-icon-theme)

## Police d'écriture

- [Operator Mono](https://github.com/willfore/vscode_operator_mono_lig) _(+ ajout de ligatures avec la police Fira Code)_

## Utilitaires

- Générateur de capture de code : [CodeSnap](https://marketplace.visualstudio.com/items?itemName=adpyke.codesnap)
- Amélioration visuelle des commentaires : [Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments)
- `!!` Gestionnaire intégré à VS Code pour Docker : [Docker](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker) _(et toutes ses extensions installées avec)_
- `!!` Tunnel entre Windows et WSL : [WSL](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)
- Formatteur de code : [Prettier - Code formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) et
- `!!` Linter Javascript couplé à Prettier : [Prettier ESLint](https://marketplace.visualstudio.com/items?itemName=rvest.vs-code-prettier-eslint)

## Syntaxes et amélioration DX

- Renommage simplifié des balises HTML/XML : [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
- `!!` Syntaxe pour les modèles Prisma : [Prisma](https://marketplace.visualstudio.com/items?itemName=Prisma.prisma)
- `!!` Syntaxe pour les fichiers Rust : [rust-analyze](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust-analyzer)
- `!!` Ajout de snippets pour Tauri : [Tauri](https://marketplace.visualstudio.com/items?itemName=tauri-apps.tauri-vscode)
- Support de la syntaxe MDX : [MDX](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx)
- `!!` Ajout d'analyse et aides pour Tailwind : [Tailwind CSS IntelliSense](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
